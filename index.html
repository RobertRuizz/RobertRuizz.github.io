T-Pot Honeypot on AWS

Cloud Security Threat Intelligence

Deployed and configured a multi-service T-Pot honeypot environment on AWS EC2 to capture and analyze real-world cyber-attack traffic. This project involved building a production-grade deception infrastructure that attracts malicious actors, logs their behavior, and provides actionable threat intelligence. By integrating the ELK stack (Elasticsearch, Logstash, Kibana), I created a comprehensive monitoring and visualization pipeline to identify attack patterns, TTPs (Tactics, Techniques, and Procedures), and emerging threats.

Key Accomplishments

  • Deployed T-Pot on AWS EC2 with proper security group configurations and network isolation
  • Configured multiple honeypot services including SSH, HTTP, and other protocols to simulate vulnerable systems
  • Integrated ELK stack for centralized logging and real-time attack visualization
  • Analyzed attack patterns, geolocation data, and adversary behavior to identify threat trends
  • Implemented automated alerts for high-severity attack indicators

Lab Network Design - NASA

Network Architecture Hardware Integration

Designed and implemented a secure, segmented lab network architecture at NASA to support research and development operations involving multiple computer interfaces and hardware systems. This project focused on creating a robust, isolated network environment that enables secure multi-endpoint access while maintaining strict separation between different research components. The infrastructure supports Linux-based NVIDIA Jetson systems and provides reliable communication frameworks for advanced hardware testing.

Key Accomplishments

  • Architected a segmented network topology with multiple isolated subnets for different research functions
  • Configured static IP addressing on Linux-based NVIDIA Jetson systems to ensure consistent connectivity
  • Developed a custom Ethernet command framework for reliable hardware-to-hardware communication
  • Implemented network security controls to prevent unauthorized access between segments
  • Documented network diagrams, IP allocation schemes, and configuration procedures for future maintenance
  • Conducted testing and validation to ensure network reliability under various operational scenarios
